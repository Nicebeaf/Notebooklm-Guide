# 贡献指南

感谢你帮助完善《NotebookLM 从基础到高阶》。

## 适合提交的内容

- 错别字、链接失效和排版问题
- NotebookLM 功能变化与官方资料更新
- 文中事实、数据和来源核验问题
- 可复现的新工作流、案例与 Prompt
- EPUB 阅读兼容性问题

## 提交 Issue

请说明：

1. 涉及的章节或文件
2. 当前内容或问题
3. 建议修改
4. 可核验的官方来源
5. 核验日期

## 提交 Pull Request

1. Fork 本仓库
2. 建立独立分支
3. 修改对应 Markdown 文件
4. 保持标题层级与现有格式一致
5. 在 Pull Request 中说明修改原因和依据

涉及功能状态、额度或地区可用性时，优先引用产品官方文档，并标注核验日期。
